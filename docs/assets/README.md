# docs/assets

Static assets for the Floke Glyph landing (`docs/index.html`).

| Path | Use |
|------|-----|
| `glyph-transparent.png` | Glyph logo (hero, nav) |
| `favicon.svg` | Browser tab icon |
| `og-cover.svg` | Open Graph preview |
| `glyph-search-2.7.js` | Optional site search bundle (glyph-s) |
| `icons/glyph.svg` | Glyph mark |

Screenshots for plugin cards live in sibling Glyph repos (`glyph-sO`, `glyph-miO`).
